redflag.imbalance.imbalance_degree(a: Union[_SupportsArray[dtype[Any]], _NestedSequence[_SupportsArray[dtype[Any]]], bool, int, float, complex, str, bytes, _NestedSequence[Union[bool, int, float, complex, str, bytes]]], method: Union[str, Callable] = 'tv', classes: Optional[Union[_SupportsArray[dtype[Any]], _NestedSequence[_SupportsArray[dtype[Any]]], bool, int, float, complex, str, bytes, _NestedSequence[Union[bool, int, float, complex, str, bytes]]]] = None) → float#

The imbalance degree reflects the degree to which the distribution of classes is imbalanced. The integer part of the imbalance degree is the number of minority classes minus 1 (m - 1, below). The fractional part is the distance between the actual (empirical) and expected distributions. The distance can be defined in different ways, depending on the method.

IR is defined according to Eq 8 in Ortigosa-Hernandez et al. (2017).

\[\mathrm{ID}(\zeta) = \frac{d_\mathrm{\Delta}(\mathbf{\zeta}, \mathbf{e})} {d_\mathrm{\Delta}(\mathbf{\iota}_m, \mathbf{e})} + (m - 1)\]
method can be a string from:

  • ‚Äòmanhattan‚Äô: Manhattan distance or L1 norm

  • ‚Äòeuclidean‚Äô: Euclidean distance or L2 norm

  • ‚Äòhellinger‚Äô: Hellinger distance, recommended by Ortigosa-Hernandez et al. (2017)

  • ‚Äòtv‚Äô: total variation distance, recommended by Ortigosa-Hernandez et al. (2017)

  • ‚Äòkl‚Äô: Kullback-Leibner divergence

It can also be a function returning a divergence.

Parameters:

  • a (array) ‚Äì A list of class labels.

  • method (str or function) ‚Äì The method to use.

  • classes (array) ‚Äì A list of classes, in the event that a does not contain all of the classes, or if you want to ignore some classes in a (not recommended) you can omit them from this list.

Returns:

The imbalance degree.

Return type:

float

Examples

>>> ID = imbalance_degree(generate_data([288, 49, 288]), 'tv')
>>> round(ID, 2)
0.76
>>> ID = imbalance_degree(generate_data([629, 333, 511]), 'euclidean')
>>> round(ID, 2)

>>> ID = imbalance_degree(generate_data([2, 81, 61, 4]), 'hellinger')
